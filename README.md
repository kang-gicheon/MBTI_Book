## 📚 구현한 기능

✅  mbti의 특징과 정보등을 위한 커뮤니티공간<br>
✅  로그인,회원가입 기능<br>
✅  12개의 질문을 통한 mbti와 mbti별 도서 추천 <br>
✅  다양한 추천 도서와 책 내용 소개 페이지 <br>
✅  mbti의 특징과 책 추천을 맡을 AI 챗봇<br>
